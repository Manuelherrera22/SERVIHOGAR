# Crear Iconos PWA

Para crear los iconos necesarios para la PWA:

## Opción 1: Usar el generador HTML

1. Abre `generate-icons.html` en tu navegador
2. Haz clic derecho en cada canvas
3. Guarda como:
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
4. Coloca los archivos en la carpeta `public/`

## Opción 2: Crear manualmente

Usa cualquier editor de imágenes (Photoshop, GIMP, Canva, etc.) para crear:

- **icon-192.png**: 192x192 píxeles
- **icon-512.png**: 512x512 píxeles

Diseño sugerido:
- Fondo con gradiente púrpura (#667eea a #764ba2)
- Icono de casa en blanco
- Bordes redondeados

## Opción 3: Usar herramienta online

- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Suba tus iconos y descarga los tamaños necesarios.
